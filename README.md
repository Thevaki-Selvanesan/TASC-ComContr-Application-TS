# TASC-ComContr-Application-TS
The type of data I chose to send was heart rate numbers. In clinical robotics systems, this type of data is sensing data, and can be used to measure the heart rate of a patient. The publisher node simulates a continuous stream of heart rate readings, similar to how a wearable biosensor would feed real-time data into a robotic system. The subscriber node acts as a monitoring system, flagging readings outside the normal range of 60 to 100 BPM as dangerous. In a real clinical setting, this type of monitoring could be integrated into a surgical or rehabilitation robot to automatically pause operation or alert medical staff when a patient's vitals become abnormal.
